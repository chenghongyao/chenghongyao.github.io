vim 速查

发表于2021-07-03|更新于2021-07-20|vimlinux
|阅读量:

快速移动光标

按键 说明
h、j、k、l 左、下、上、右
w、b 下一个单词、上一个单词
W、B （包括特殊负号的）下一个单词，上一个单词
{、} 下一个、上一个段落

修改

按键 说明
gU、gu 大写、小写转换
r<c> 字母修改为<c>

多窗口命令

  • :vs <file> 垂直分栏打开文件
  • :sp <file> 水平分栏打开文件
  • :vs 当前窗口垂直分割
  • :sp 当ian窗口水平分割
  • :only：保留当前窗口，关闭其他
  • <c-w>[j|k|h|l] 光标向上下左右的窗口移动
  • <c-w>w 光标移动到下一个窗口
  • <c-w>p 光标移动到上一个窗口
  • <c-w>t 光标移动到左上角窗口
  • <c-w>b 光标移动到右下角窗口
  • <c-w>L 当前窗口移动到最左端
  • <c-w>[r|R] 向右或向下放交换窗口
  • 修改尺寸
    • <c-w>+ 将当前窗口的高度增加一行
    • <c-w>- 将当前窗口的高度减少一行
    • <c-w>= 将所有窗口的高度等分
    • <c-w>< 将当前窗口的宽度减少
    • <c-w>> 将当前窗口的宽度增加
    • <c-w>= 将所有窗口的高度等分
    • res[ize] num 设置窗口行数
    • res[ize] +num 窗口增加num行
    • res[ize] -num 窗口增加减少行
    • vert[ical] res[ize] num 设置窗口列数
    • vert[ical] res[ize] +num 窗口增加num列
    • vert[ical] res[ize] -num 窗口减少num列

其他

  • Ctrl-Z 暂时退出 vim，回到终端,命令行输入 fg 返回 vim
按键 说明
zc、zo 折叠（close）、展开（open）当前代码段
zC、zO 折叠（close）、展开（open）当前代码段
